This is a soundcloud client for the command line that supports searching and streaming soundcloud songs from the command line.


Requirements:
--------------
VLC media player
Python 2.6+


Commands:
---------
./soundcloud.py 'artist' track
./soundcloud.py 'artist-album' playlist

Controls:
---------
Enter = Stream
SPACE = Download
Q = Quit
